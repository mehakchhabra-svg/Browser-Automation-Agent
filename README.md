# Browser-Automation-Agent
This project is a Python-based automation agent that logs into a personal email account using a mail server connection, composes an email based on user input, and sends it to the recipient's email address.
TOOLS USED:
- Python
- smtplib
- email. Message
- Steps to execute the agent:
- Check if python is installed in your system, if not install Python and proceed with following steps.
- Open command prompt
- cd Desktop
type nul > email_agent.py
dir email_agent.py
notepad email_agent.py
Paste the code of the task
py email_agent.py
Enter the details and see the output.
